## How to Use

1. **Open the app**: Simply open `index.html` in your web browser
2. **Load a video**: Use the file input to select a video from your device
3. **Play the video**: The video will automatically start playing after selection, or use the built-in controls

## Supported Video Formats

The app supports all video formats that your browser supports, including:
- MP4
- WebM
- Ogg

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - JavaScript for file loading and keyboard controls

## Customization

You can customize the default video by replacing the `src` attribute in the `<video>` tag, or simply use the file input to load any video from your device.
